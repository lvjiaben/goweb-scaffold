# v0.9.0-rc.1

## 版本定位

这是 `goweb-scaffold` 的正式 release candidate，不是 final stable。

## 本版本主要能力

- admin / user 双端基础应用
- 自定义 RBAC
- 系统模块与附件模块
- codegen 生命周期：
  - lock
  - export / import
  - migrate-source
  - batch
  - snapshot
  - check-breaking
- 当前只生成 admin CRUD，不生成 user 页面

## 本版本不包含

- user 页面生成
- 第十一阶段真实业务试跑结论
- 最终稳定版承诺

## 升级建议

- 旧 lock/export 先走 `migrate-source`
- 再执行 `check-breaking`
- 确认结果后再 `regenerate`

## 兼容关系

- 兼容 `goweb-core v0.9.0-rc.1`
- 当前 template version：`v7`
- 支持 legacy source migration：`v5` / `v6` / `v7`
